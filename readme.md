# Лабораторная работа №2 — Волк и 7 козлят  
**Межпроцессное взаимодействие в Linux**  
Родственная конфигурация «один ко многим»

Реализовано три механизма IPC:
- POSIX Message Queues (`mq`)
- Именованные каналы (`fifo`)
- Unix domain sockets (`sock`)

### Что сделано
- 7 дочерних процессов-козлят (fork)
- Отдельный поток на каждого козлёнка + `std::barrier`
- Чистое завершение и удаление всех ресурсов
- Сборка через CMake + `build.sh`
- `-Wall -Werror` — компилируется без предупреждений

### Как собрать и запустить

```bash
# 1. Перейти в папку с лабой
cd "/.../..."

# 2. Полная пересборка (обязательно после любых изменений)
./build.sh
# или вручную:
# rm -rf build && mkdir build && cd build && cmake .. && make -j

# 3. Запуск любой из трёх реализаций
./build/host_mq      # очереди сообщений
./build/host_fifo    # именованные каналы
./build/host_sock    # сокеты
